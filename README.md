# 三国演弈 v2.0 — 冷兵器沙盒模拟器

> 配置驱动的三国自动战斗沙盘。选将 → 布阵 → 拖拽出生点 → 开战 → 观战 → 分析。

## 快速开始

```bash
pnpm install
pnpm dev
```

## 十大势力

魏 / 蜀 / 吴 / 群雄 / 董卓 / 袁绍 / 西凉 / 荆州 / 益州 / 晋 — 每阵营 20 人

14 名神将可选开启。任意将领可单独强化（☆按钮，HP×2.5 攻×2 防×1.8）。

## 联盟系统

支持跨阵营联盟，联盟内不互攻、共享士气光环、战吼互相生效、投降/溃败判定认盟友：
- 赤壁之战：蜀+吴 vs 魏
- 樊城之战：蜀 vs 魏+吴
- 下邳之战：魏+蜀 vs 吕布
- 西凉叛乱：西凉+群雄 vs 魏

## 出生点拖拽

布阵后地图上显示阵营圆圈，拖拽改变出生位置，开战后展开为具体将领阵型。

## 四层 AI

```
第1层: 指挥风格 — 8种(稳扎稳打/全军突击/以逸待劳/两面夹击/斩首行动/化整为零/群龙无首/兵败如山)
       指挥官死后退化, 新将领可接替
第2层: 高级AI — 战局5阶段判断 + 阵营战术(蜀=骑兵突击/吴=火攻/袁绍=防守/益州=游击)
       地形利用: 桥头+20%防/山旁弓兵+15%攻/森林游击+18%攻
第3层: 指挥官AI — 集火/撤退/保护指令, 攻城模式特殊行为
第4层: 兵种AI — 弓兵风筝(退向友军)/骑兵绕后(不冲人堆)/盾兵护弓/枪兵反骑/步兵清残
```

## 六维属性

| 属性 | 影响 |
|------|------|
| 统 | 阵型协同、攻防加成 |
| 谋 | 目标选择、技能触发、远程攻击+40%(谋略100) |
| 政 | 士气恢复、抗疲劳、补给线、治疗加成 |
| 勇 | 近战+40%、横扫(≥85打多人)、闪避(≤18%)、反击(≥88)、威慑、背水+60% |
| 功 | 初始士气、威慑减防 |
| 德 | 士气光环160px、战吼+15士气、抗溃败 |

## 战斗机制

**武力型**: 横扫 / 闪避 / 反击 / 威慑 / 破阵 / 背水一战 / 骑兵冲锋护盾(25tick士气免疫)

**战术型**: 侧击+35% / 冲锋动量+80% / 森林伏击+25-50% / 单挑 / 弓兵齐射 / 盾墙 / 枪兵架枪 / 战吼 / 斩将(按距离衰减) / 投降 / 士气连锁

**兵种克制**: 骑克弓1.4x / 枪克骑1.35x / 盾克弓1.3x / 弓克枪1.2x

## 士气系统

- 撤退线22 / 溃败线6 / 溃败恢复线38
- 纪律减少士气损失(纪律95=-38%)
- 指挥官集结180px加速恢复
- 每tick最多掉20(防雪崩)
- 连锁崩溃5% / 70px / -8士气
- 骑兵冲锋后25tick士气免疫+10士气
- 背水一战: HP<25%时攻+40%防+25%

## 地形
