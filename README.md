# 三国演弈 v2.0 — 冷兵器沙盒模拟器

> 配置驱动的三国自动战斗沙盘。选将 → 布阵 → 开战 → 观战 → 分析 → 再来一把。

## 快速开始

```bash
pnpm install
pnpm dev
```

## 项目概览

## 十大势力

魏 / 蜀 / 吴 / 群雄 / 董卓 / 袁绍 / 西凉 / 荆州 / 益州 / 晋 — 每阵营 20 人

14 名神将可选开启，任意将领可单独强化（☆按钮）

## 联盟系统

支持跨阵营联盟，联盟内不互相攻击，共同作战：
- 赤壁之战：蜀+吴 vs 魏
- 樊城之战：蜀 vs 魏+吴
- 下邳之战：魏+蜀 vs 吕布

## 六维属性

| 属性 | 影响 |
|------|------|
| 统 | 带兵效率、阵型协同 |
| 谋 | 目标选择、技能触发、远程攻击加成 |
| 政 | 士气恢复、抗疲劳、补给线 |
| 勇 | 近战输出、横扫、闪避、反击、威慑 |
| 功 | 初始士气、威慑减防 |
| 德 | 士气光环、战吼、抗溃败 |

## 四层 AI 架构

```
第1层: 指挥风格 — 8种风格，指挥官死后退化
第2层: 高级AI — 战局判断+阵营战术+地形利用
第3层: 指挥官AI — 集火/撤退/保护
第4层: 兵种战术AI — 弓兵风筝/骑兵绕后/盾兵护弓/枪兵反骑
```

### 指挥风格（指挥官性格决定）

稳扎稳打 / 全军突击 / 以逸待劳 / 两面夹击 / 斩首行动 / 化整为零

指挥官战死 → 群龙无首或兵败如山 → 新将领可接替指挥

### 阵营战术

蜀/西凉=骑兵突击 / 吴=火攻 / 袁绍/荆州=防守消耗 / 益州=游击 / 群雄/董卓=暴力进攻 / 魏/晋=均衡

## 战斗机制

### 武力型

横扫（武力≥85多目标）/ 闪避（最高18%）/ 反击（武力≥88）/ 威慑（武力≥88周围敌人-8%攻）/ 破阵 / 背水一战（HP<30%攻+60%）/ 骑兵冲锋护盾（25tick士气免疫）

### 其他

侧击 / 冲锋动量 / 森林伏击 / 单挑 / 弓兵齐射 / 盾墙 / 枪兵架枪 / 战吼 / 斩将 / 投降 / 士气连锁 / 纪律士气护甲

### 兵种克制

骑兵克弓兵(1.4x) / 枪兵克骑兵(1.35x) / 盾兵克弓兵(1.3x) / 弓兵克枪兵(1.2x)

## 士气系统

- 撤退线 22 / 溃败线 6 / 溃败恢复线 38
- 纪律减少士气损失（纪律95=-38%）
- 指挥官集结 180px 加速恢复
- 每 tick 最多掉 20 士气（防雪崩）
- 连锁崩溃 5% 概率 70px 范围

## 阵法（开战前选）

鹤翼阵（弓兵+射程攻击）/ 锥形阵（骑兵+冲锋速度）/ 方圆阵（防御+士气）/ 鱼鳞阵（攻防均衡）/ 雁行阵（机动）/ 长蛇阵（高速）
