# 三国演弈 — 冷兵器沙盒模拟器

> 配置驱动的三国自动战斗沙盘。选将 → 布阵 → 开战 → 观战 → 分析 → 再来一把。  
> 三国版电子斗蛐蛐 / 自动战斗沙盒 / 吃鸡推演器。

## 快速开始

```bash
pnpm install
pnpm dev
```

浏览器打开 http://localhost:5173/

## 核心玩法

1. **选将** — 左侧面板勾选参战将领（200+ 名将领，10 大势力 + 14 名神将）
2. **选图** — 15 种地图模板，从大平原到栈道到赤壁
3. **选模式** — 阵营对抗 / 混战 / 攻城
4. **配置** — Seed、速度 0.5x~8x、10 项玩法开关
5. **开战** — 观看 AI 驱动的自动战斗推演
6. **交互** — Canvas 缩放/拖拽、点击单位查看六维雷达图详情
7. **分析** — 战后排名、MVP、阵营统计、击杀/输出/承伤
8. **批量** — 跑 20/50/100 次看胜率分布
9. **战史** — 本地自动保存对局记录

### 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开战 / 暂停 / 继续 |
| `0` | 0.5x 速度 |
| `1` `2` `3` `4` | 1x / 2x / 4x / 8x 速度 |
| `R` | 重开（新 Seed） |
| `ESC` | 取消选中单位 |
| `双击画布` | 重置缩放 |

## 战斗系统

### 武力型战斗机制

高武力将领不再只是"攻击力高"，而是拥有完整的近战体系：

| 机制 | 触发条件 | 效果 |
|------|---------|------|
| **横扫** | 武力≥80 近战 | 同时命中 1-3 个 45px 内敌人，55% 伤害 |
| **闪避** | 速度+武力 | 最高 25% 概率闪避近战攻击 |
| **反击** | 武力≥85 | 被近战打时 30% 概率反击（35% ATK） |
| **威慑** | 武力≥85 | 65px 内敌人攻击力 -12% |
| **破阵** | 武力>敌方统帅 | 削减敌方阵型防御加成 |
| **背水** | HP<30% | 武力×0.6% 加成（武力 100 = +60%） |

### 其他战斗机制

| 机制 | 触发条件 | 效果 |
|------|---------|------|
| 侧击 | 从背后攻击 | +35% 伤害 |
| 骑兵冲锋 | 连续移动 60px+ | 首击最高 +80% |
| 森林伏击 | 从森林攻击非森林目标 | +25-50% 伤害 |
| 武将单挑 | 两个武力 80+ 互攻 | 独立战斗回合 |
| 弓兵齐射 | 3+ 弓兵 80px 内 | 攻击力 +20-45% |
| 盾墙阵型 | 2+ 盾兵 50px 内 | 防御 +30-50% |
| 枪兵架枪 | 敌骑兵 120px 内 | 防+40% 攻+25% |
| 战吼鼓舞 | 德 80+ | 友军回士气+攻击 |
| 斩将夺旗 | 击杀指挥官 | 全阵营士气 -20 |
| 临死一搏 | 武力 75+ 阵亡 | AoE 伤害 |
| 投降 | 孤立+残血+低士气 | 退出战斗 |

| **长坂坡** | 宽河只有一座桥，两侧密林 |

### 地形类型

| 地形 | 通行 | 效果 |
|------|------|------|
| 平原 | 可 | 无修正 |
| 森林 | 可 | 速度 0.55x, 防+15, 视野-50%, 可放火 |
| 山脉 | 不可 | 完全阻挡，靠近+15% 防 |
| 河流 | 不可 | 阻挡移动，近战无法穿越 |
| 城墙 | 不可 | 类似山脉，砖块纹理 |
| 桥梁 | 可 | 速度 0.8x |
| 渡口 | 可 | 速度 0.5x, 防-10 |

### 地形互动

- **森林起火**：火计命中 → 蔓延 → 风天加速/雨天减缓 → 烧光变平原
- **河水泛滥**：强雨时河流扩展
- **山地掩护**：靠近山脉 +15% 防

## 游戏模式

### 阵营对抗
最后存活的阵营获胜。

### 混战
最后存活的将领获胜。

### 攻城
- 人数最少的阵营守城，其余攻城
- 城墙 + 4 城门 + 4 座箭塔（HP 500, 射程 130, 伤害 35）
- 攻方胜：占领城心 200 tick 或消灭守方
- 守方胜：消灭攻方或坚守 4000 tick

## 项目结构

```
src/
  types/                # 类型定义
  config/
    generals.ts         # 80 名四大阵营将领 + 14 神将
    factions.ts         # 120 名六大边缘势力将领
    factionDisplay.ts   # 统一阵营颜色/名称常量
    skills.ts           # 12 种技能
    balance.ts          # 平衡参数、地形、兵种克制、武力机制
    campaigns.ts        # 12 场历史战役
  engine/
    BattleEngine.ts     # 主引擎
    systems/
      movement.ts       # A* 寻路 + 墙壁滑行 + 卡住检测
      target.ts         # 视线 + 评分索敌
      attack.ts         # 伤害 + 闪避 + 横扫 + 反击 + 威慑 + 破阵
      morale.ts         # 士气 + 连锁溃败
      skill.ts          # 技能触发
      victory.ts        # 胜负判定
      weather.ts        # 天气
      tacticalAI.ts     # 兵种 AI
      commanderAI.ts    # 指挥官 AI
      formations.ts     # 齐射/盾墙/架枪/战吼/投降
      combatMechanics.ts # 单挑/冲锋/伏击/斩将
      terrainInteraction.ts # 火焰/泛滥/山地
      battlefield.ts    # 补给/毒圈
      siege.ts          # 攻城箭塔/占领
      traits.ts         # 被动特质
    utils/
      random.ts         # Mulberry32 PRNG
      math.ts           # 向量数学
      mapgen.ts         # 15 种地图模板
      unitFactory.ts    # 将领 → 战斗单位 + 出生点
      pathfinding.ts    # A* + Bresenham LOS
      vfx.ts            # 弹道/尾迹/单挑高亮
      viewport.ts       # 缩放/平移
      history.ts        # localStorage 对局历史
  store/gameStore.ts    # Zustand 状态
  components/RadarChart.tsx
  features/
    battle/             # Canvas、控制栏、日志、小地图、设置、Kill Feed
    generals/           # 将领配置面板
    stats/              # 结果、批量、历史
  hooks/useKeyboardShortcuts.ts
```

## 技术栈

Vite + React 18 + TypeScript + Zustand + Tailwind CSS 4 + Canvas 2D  
纯前端，无后端依赖，localStorage 持久化。
